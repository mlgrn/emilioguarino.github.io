WavPool Chat

ReactJS & Firestore Database

WavPool Chat Is A ReactJS Based Chat App With Firebase Backend

Use your Google account to login to a chat app built with ReactJS, Material UI Icons, npm reactjs-popup, and Firestore database on Firebase. State is managed by wrapping the app inside the React Context API <StateProvider>

User can add or remove rooms, leave messages, or delete their own messages.
