Tenzies

React Dice Game

What Is Tenzies?

Tenzies is a dice game where the player chooses a die face to match, for example #6, and keeps rolling the other dice until they all match.

How was this made?

ReactJS, HTML, and CSS

Where can I play?

You can play here (desktop browser recommended)
